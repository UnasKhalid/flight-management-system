# Flight Management System

**Course:** ENSF 337 – Programming Fundamentals, University of Calgary  
**Term Project – Fall 2025**

---

## Overview
A C++ console application that simulates a small airline flight management system. Users can manage flights, view seat maps, add/remove passengers, and save updated data.

---

## Features
- Load and save flight and passenger data from text files  
- Browse and select flights  
- Display flight seat maps with available/occupied seats  
- Add or remove passengers  
- Menu-driven interface for easy interaction  

---

## Technologies
- **Languages:** C++  
- **Concepts:** Object-Oriented Programming, Classes, Vectors, File I/O
